# dotfile setup
My dotfile setup for zsh, tmux, and vim. 

## requirements
* bash
* git
* curl
* zsh
* tmux
* vim
* powerline (optional)

## installation
`git clone https://github.com/joecochran/dotfiles.git`
`cd dotfiles`
`./install.sh
